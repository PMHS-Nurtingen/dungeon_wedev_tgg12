# Was sind semantische Elemente?
Semantische HTML Elemente sind zum Beispiel das **header**, **main** oder **footer** Element. Also z.B.

```html
<header>
    Dies ist ein Inhalt innerhalb der semantischen header-Elements.
</header>
```
